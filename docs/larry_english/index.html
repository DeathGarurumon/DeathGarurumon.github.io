Larry 的英语学习真理之语法篇笔记

0.前言
#

我们已经学习了多年的英语，甚至从学前开始就接触英语，我们不断的学习这门语言，但仿佛根本学不完也学不透。明明语法就那么几个，明明句式就那么几套，明明词汇都认识，为什么就是读不懂句子、读不懂文章呢？

很可能是只知其表，不知其里；只知其名，不知其实。我们不妨提纲挈领，将各类语法归宗明义，简单本质地理解英语句子和语法。英语语法的本质就是不同词性间的相互修饰关系。

这里是 Larry 的英语学习真理 系列的 语法篇 笔记，如果对 Larry 感兴趣可以去 BiliBili 搜索 Larry想做技术大佬。

1.词性
#

1.三大成分
#

  1. 名词性成分（red）：名词，名词性从句，不定式，动名词

  2. 形容词性成分（yellow）：形容词，形容词性从句，不定式，分词，介词短语

  3. 副词性成分（green）：副词，副词性成分，不定式，介词短语

  4. 动词：谓语

2.修饰关系
#

  1. 形容词修饰名词

  2. 副词修饰形容词、动词、其他副词

3.连词
#

三大成分互相修饰形成简单句，简单句通过连词变为复杂的从句结构

4.介词
#

  1. 介词用法只有一种：介词＋名词性成分 = 介词短语

  2. 介词短语可以是形容词或副词性成分

  3. 形容词性成分的介词短语：一般是后置定语，修饰名词

  4. 副词性成分的介词短语：修饰动词，形容词和副词性成分

  5. 特殊情况：介词＋副词等构成介词短语

2.简单句
#

1.主谓宾定状补
#

  1. 主语：名词性成分（句子的主体话题 or 动作的发出者）

  2. 谓语：动词

  3. 宾语：名词性成分（谓语动词的执行对象）

  4. 定语：形容词性成分（修饰名词性成分）

  5. 状语：副词性成分（修饰动词，形容词，其他副词性成分）

  6. 补语：形容词 or 名词性成分（补充说明的额外成分）

    补语分为 宾语补语 和 主语补语：可理解为 A is B （系表结构）

2.简单句的五种句型
#

  1. 主谓（SV）：不及物动词

  2. 主系表（SVP）：系动词（=）

  3. 主谓宾（SVO）：及物动词

  4. 主谓双宾（SVOO）：及物动词（直接宾语通常是物品，间接宾语通常是人；可以用 for 或 to 改写）

  5. 主谓宾补（SVOC）：及物动词（A is B；主补或宾补）

3.连词
#

  1. 简单句＋连词 = 并列结构 or 主从结构

  2. 疑问词的词性：

3.从句
#

  • 从句也是简单句，属于五种基本句型的一种，可以找出主谓宾定状补的成分

  • 独立句子变成从句的步骤：调语序（调整成陈述语序），加连词（陈述句用 that，一般疑问句用 whether，特殊疑问句用 原本的疑问词），把连词置于从句句首

1.名词性从句
#

  • 陈述句、一般疑问句、特殊疑问句 代替简单句中的名词性成分变成 主语从句、宾语从句、表语从句 之一

  1. 同位语：针对抽象名词的 名词性 解释说明（that 在同位语从句中不担当成分，而在定语从句中担当主语或宾语）

  2. 形式主语：用 it 替换主语位置，将主语位置的从句放在句尾

  3. 形式宾语：用 it 替换宾语位置，将宾语位置的从句放在句尾，且必须这么做

  4. that 的省略：在不影响理解时可以省略

  5. whether 和 if 的互换问题：表达 是否 的意思时，whether 比 if 更加通用，无脑 whether 即可（if 可能解释成 如果 造成歧义，带有 or not 的固定搭配只能用 whether，介词后也只能用 whether）

2.形容词性从句
#

  • 形容词性从句要修饰一个名词性成分，可有可无

  1. 关系词和先行词：关系词是从句中的连词，先行词是主句中的形容词性从句修饰的名词

  2. 关系词的作用：重复指代先行词（且与先行词单复数，主格宾格等性质保持统一）；连词作用

  3. 主语关系代词：who（某人），which（某人），that（某人或某物）；指代某人时，更常用 who，指代某物时，更常用 that

  4. 宾语关系代词：who，whom，which，that；做形容词性从句中宾语时，这些关系代词可以省略（若从句较长，且宾语原位置离关系代词较远，则不要省略）

  5. 介词宾语关系代词：who，whom，which，that；做介词后的宾语时，分为介词后置和介词前置；介词后置就是介词在从句中，介词后的宾语应为关系代词，介词前置是介词在主句中，介词后的宾语为关系代词（只有 介词＋whom 和 介词＋which）；注意，动词短语（如 look for）不能拆开将介词前置，因为如果拆开，look 和 look for 的含义不同

  1. 状语关系副词：where（地点），when（时间），why（原因）；where 与 in which 等价（where 更常用）； when 与 on which 等价；why 与 for which 等价

  2. 定语关系限定词：whose（人或物都可以），which；whose 与 of which 等价；which fact 可以作为关系词

  3. 补语关系代词：which，that；关系词在句子中做补语

  4. 限制性和非限制性：非限定性定语从句的判断：有逗号，关系词只能用 which，who，where，when（特殊情况可以用 that）；限制性是一种筛选，从千千万万个物品中特指一个，所以当专有名词做先行词，即只有一个的物品或人不能使用限制性定语从句；非限制性定语从句的先行词更加多样（可以是一个名词，也可以是一件事/一句话）

3.副词性从句
#

  1. 时间状语从句：when（whenever），while，as，before，after，since，until，一…就…（as soon as，immediately，the moment/minute/second，no sooner…than，hardly…when…，scarcely…when…）

    when：时间点或时间段（从句中的动词可以是短暂性，也可以是持续性）

    while：时间段（从句中动词只能是持续性）

    as：随着…，当…，一边…一边；

    before，after：之前，之后

    since：自从

    until：保持某种状态 until 某个时间点；主句持续性动词肯定式 or 主句短暂性动词否定式＋until＋时间点

    Not until＋时间点＋主句短暂性动词（倒装句）

    as soon as/immediately/the moment/the minute/the second：一 从句 就 主句

    no sooner…than，hardly…when…，scarcely…when…：前句为从句（倒装），后句为主句

  2. 地点状语从句：where（具体地点，抽象地点）

    where there is a will，there is a way.

  3. 原因状语从句：because，for，as，since

    since：既然

    because 的否定转移：not because 中的 not 要提前到主句；在语义通顺的情况下，主句中的否定表示 不是因为 的意思

  4. 目的状语从句：so that，in order that，that，so

  5. 结果状语从句：so…that…，such(…可省略)that…，so that（so 是副词，such是形容词）

  6. 条件状语从句：if，unless，once，as long as，so far as，in case 等

  7. 让步状语从句：though，although，even if，while，no matter＋疑问词（how） = 疑问词＋ever（however）（不能用 but，可以用 yet 或 still）；介词短语：despite，in spite of；as 引导的倒装句

  8. 比较状语从句：as…as 如同，好像，no more than，no less than，more than，less than；Mike is as A as B：Mike 既 A 又 B；

4.非谓语动词
#

1.不定式（名词，形容词，副词）
#

  1. 不定式有不同时态：一般时（谓语动词发生在不定式动作之前或同时），进行时（谓语动词发生时，不定式动作正在进行），完成时（不定式动作在谓语动词发生之前），完成进行时

  2. 不定式有主动和被动：进行时 和 完成进行时 无被动

  1. 使役动词和感官动词后续的不定式省略 to：have，make，let，watch 等

  2. 插入语：to be frank，to be fair，to begin with，to conclude，to be honest，to tell the truth，so to speak，to say the least of…

  3. 逻辑主语和逻辑宾语：可以使用 for、with 或 of 结构，给不定式赋予逻辑主语或逻辑宾语，否则逻辑主语或宾语就是句子的主语或宾语；表语 of xxx 适用于可以写成 “ xxx is 表语 ” 形式的表语（例如：it’s wise of you to do sth. 表语是wise，可以写成 you are wise，所以可以用 of）

2.动名词（名词）
#

  1. It is no good/no use/a waste of time＋doing sth 做某事没用

  2. There is no point/use/good (in) doing sth 做某事没意义，没必要

  3. There is no doing sth 不可能做到某事

  4. 动名词作宾语：admit，advise，avoid，consider，deny，like，dislike，enjoy，feel like，finish，forgive，give up，can’t help，imagine，involve，mention，mind，miss，permit，prohibit，resist，risk，suggest 等

  5. have difficulty/trouble/problems/a hard time (in) doing sth 做某事遇到困难

  6. can’t help/resist/keep from doing sth 情不自禁做某事

  7. 逻辑主语：动名词前加物主代词（my，his，her，their，sb’s 等），所有格；逻辑宾语：动名词前加物主代词或宾语都可以，即所有格或宾格（his，him）；当逻辑主语太长，或是不定代词时（someone，somebody 等），用普通格即可

  8. 时态和主被动：

  1. 主动表被动：need，deserve，want，require，won’t bear；worth；past，beyond

  2. 动名词和不定式的区别：动名词泛指所有人，强调一般时态或过去时态；不定式特征某个人（通常是主语），强调下一次的未来时态

3.分词（形容词，副词）
#

  1. 现在分词：主动的，正在进行的动作；主动的，一般性的动作

  2. 过去分词：被动的，一般性的动作（及物动词）；主动的，一般性的动作（不及物动词，如 fallen leaves）

  3. 逻辑主语需要合理，可以选择独立分词结构，即在分词前加一个逻辑主语（名词＋分词 或 with＋名词＋分词）

  4. 固定结构（不必考虑逻辑主语）：generally speaking，strictly speaking，judging from，talking of，considering，granting

  5. 分词作状语：条件，时间，原因，让步，结果（表达结果状语的分词要放到句末，而不能放句首，因为有可能产生歧义），方式；分词作状语时，可带有连词（可理解为从句变成分词时，保留了连词）

  6. 分词作表语：当作形容词使用，可以被副词修饰，也可以有比较级

  7. 分词作定语：

    现在分词作定语，其修饰的名词是分词动作的发出者，构成主谓关系，且有正在进行的含义；一般来说，现在分词放在修饰的名词前，若放在名词后，则表示强调该分词动作

    过去分词作定语，其修饰的名词是分词动作的承受者，构成动宾关系，表示被动行为（及物动词）；不及物动词的过去分词仅表示完成，而不表示被动

    单个分词作定语，一般前置；分词短语作定语，一般后置

  8. 分词作补语：一般在感官动词之后，一般后置

4.非谓语动词与从句的转换
#

  1. 名词性从句 可以转换为 名词，动名词，不定式

  2. 形容词性从句 可以转换为 分词，介词短语

  3. 副词性从句 可以转换为 分词，不定式，介词短语

  4. 非谓语动词表意方面的准确性更差，对于时态也更模糊，需要上下文

5.动词的时态
#

1.动词时态
#

  1. had been doing：过去一段时间内持续进行，未完成

  2. have been doing：至今为止持续进行，未完成

  3. was/were doing：当时正在进行，未完成

  4. be doing：现在正在进行，未完成

  5. done：过去的动作，已完成

  6. do：经常性，重复性动作，表示能力或状态

2.动词的分类
#

  1. 状态动词（states）：（have，know，love，want，belong）表示状态，一般不用于进行时；在句子中，活动动词可能体现状态动词的意思；一般现在时表示永久或反复的状态，现在进行时表示暂时的状态（I am living in Harbin. 我暂时居住在哈尔滨）

  2. 活动动词（activities）：（walk，swim，write，read）表示活动，动作没有起点和终点 或 起点和终点并不明确

  3. 过程动词（accomplishments）：（build，persuade，prepare）表示一个时间段，动作有起点和终点，一般不使用一般现在时

  4. 瞬间动词（achievements）：（kill，drop，catch，arrive）表示一个时间点，动作瞬间发生并结束，一般不使用一般现在时（当使用 一般现在时 或 过去时 时，可以表示一种习惯）

3.时间状语分类
#

  1. 时间点：tomorrow morning，three years ago（瞬间动词）

  2. 定位时间段（确定的某段时间）：since 2023，during the first half of last year（过程动词）

  3. 不定位时间段（不确定的某段时间）：for one year，for three days（活动动词）
