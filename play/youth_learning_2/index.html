前言
#

随着网购的普及，生活中常常需要在网上购买具有各类尺寸的物品，挑选合适的尺寸就变得十分重要。由于无法试穿或试用，退换货也很麻烦，因此一次性选择正确的尺寸是十分必要的。

先介绍几个常见的尺寸，日常生活中很少能突然拿出来一个尺子，但量东西的情景又很多。因此介绍几个常见物品的尺寸，可以代替尺子使用。

  • 银行卡、身份证等标准卡

    国际标准的卡片大小是统一的：85.60mm x 53.98mm

  • A4 纸和 B5 纸

    纸张的规格也是固定的：A4 纸是 210mm x 297mm，B5 纸是 176mm x 250mm

  • 钱币

    纸币的规格也是统一的，但不同面值的纸币大小不同，但有规可循。100 元：155mm x 77mm；50元：150mm x 70mm；20元：145mm x 70mm；10元：140mm x 70mm；5元：135mm x 63；1 元纸币：130mm x 63mm。观察可以发现，每大一个面值，长度都会增加 5mm，因此记住一个就可以推导出其他的。

  • 手

    成人的一拃约为 20cm，一拃就是伸开拇指和中指后它们之间的距离。可以提前量好伸开的角度和长度，例如直角，便于观察。

  • 手机

    每种手机尺寸都有差异，可以提前测量好手机尺寸，或者根据型号搜索厂家提供的数据，需要时即可应用。

2. 钻戒大小
#

在选购戒指时，合适的圈口尺寸至关重要。尺寸不当，戒指要么佩戴时紧勒手指，带来不适，影响血液循环；要么过于宽松，容易在不经意间滑落丢失。

销售戒指时，有的采用大陆版尺寸标准，有的采用港版尺寸标注，购买时应该问清尺寸（直接问内径即可）

如何测量自己手指的尺寸呢？选取一根细线或纸条，绕手指一周，用笔标记交汇处，随后将线拉直用尺测量。为确保测量精准度，可以多次测量取平均值。珠宝店通常有专业测量环，可以去店内测量，这种方式最准确。

在傍晚时和夏季时手指会变粗，选择戒指时应该注意。

3. 电视和电脑屏幕
#

购买电视时，主要考虑沙发离电视的距离，购买合适的尺寸即可，太大不容易看到完整画面，太小又看不清楚。

购买电脑时，显示器离椅子最多也就是 80cm，因此主要考虑需求。如果主要用来打游戏，23寸左右的最为适宜；办公用 27 寸左右比较舒服；如果需要写代码、剪视频等重度需求，最好考虑副屏。

4. 蛋糕尺寸
#

在订做蛋糕时，通常需要选择尺寸，太大了吃不了，太小了又不够吃；若是自己做蛋糕，不同尺寸的用料又不尽相同。这里介绍一下蛋糕尺寸与换算比例。

圆形蛋糕的尺寸是直径，方形蛋糕的尺寸是对角线长度。一英寸约为 2.54cm，一磅约为 453g。若是自己做蛋糕，尺寸的平方比就是用料比，且每大一个尺寸，温度都要降低 5-10 度，烤焙时间延长 10-15 分钟。（因为用料比=体积比=底面积比=尺寸的平方比）

5. 行李箱尺寸
#

出差旅行必备的就是行李箱，坐飞机托运和手提都有一定的要求，有时我们的行李也对行李箱的大小有一定要求，因此需要选择好行李箱尺寸。

一般来说，国内航班要求：手提行李体积 ≤ 20x40x55cm，托运行李体积 ≤ 40x60x100cm；国际航班要求：手提行李三边之和 ≤ 115cm，托运行李三边之和 ≤ 158cm

6. 相片尺寸
#

我们在填写各种资料、办理各种证件时经常需要照片，如何判断我们的照片大小是否标准就至关重要。

通常证件照为蓝底、白底、红底；蓝底一般用于个人证件（毕业证、工作证、简历等）；红底一般用于公家证件（保险、暂住证、结婚证等）；白底一般用于国际证件（签证、护照等）

7. 纸张尺寸
#

日常生活中最常见的就是纸张，打印时经常也有要求 A4、B5 等等大小的纸，如何区分呢？

常见的几种规格是 A、B、C、K 系列。

A 和 B 系列的长宽比都是 \(\dfrac{\sqrt{2}}{1}\) ，A0 的尺寸为：841mm x 1189mm，B0 的尺寸为 1000mm x 1414mm。数字每增加 1，纸张就对折一次。

C 系列主要用于信封制作，C0 的尺寸为 917mm x 1297mm。

K 系列的纸张是中国传统的开本尺寸，一般书籍中会注明多少开，指的就是这个。分为正度和大度，正度的全开尺寸为 787mm x 1092mm，大度的全开尺寸为 880mm x 1230mm，一般的书籍用的都是正度。
