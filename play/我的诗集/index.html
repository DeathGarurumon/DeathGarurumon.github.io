打油诗与定场诗诗集（包括作者原创和收集摘录的）

绝命原创（由新到旧）
#

天外飞仙——作于飞机上
#

恍惚仿佛隔世间，轻舟已过万重山。

云山雾海何人见，不可亵玩惟远观。

良辰美景度经年，古人何处赏今颜。

此番腾云天际线，胜似神仙落九天。

叹水课
#

低声细语无人爱，照本宣科本无奈。

忽闻下课铃声骇，师生无言别见怪。

教师赞
#

琼畟入墨盅，已是日暮时分。

巷尾点香球，化作月夜星辰。

夜半犹灯火，感念用心良苦。

便是践红毹，难忘诲人之恩。

君莫叹
#

君莫叹，往事竟如烟。

自古多情难厮守，人生难得一成全。

相逢便是缘。

约会
#

值此秋末晚阳天，偷得浮生半日闲。

闲情信步赏美景，嬉笑欢闹拥芳筵。

美食美景有何恋？只因爱人在身边。

愿君两情长相守，头上花枝岁岁年。

黎明飞仙（其一）——夜半作于飞机上
#

过万重山薄雾酣，山薄雾酣云淡天。

云淡天轻舟已过，轻舟已过万重山。

黎明飞仙（其二）——夜半作于飞机上
#

于黎明前路漫漫，前路漫漫其修远。

漫其修远何人处？何人处于黎明前？

冬夜期末长
#

又逢雪降，举笔环望，心彷徨。

凝望手中书，只感冬寒期末长。

遥想亲朋尽归乡，更添无限凄凉。

吾身依旧在，心已悄然飘远方。

夜读——留校备战夏令营
#

长安夜上雨初停，孤单单，冷清清。

偌大夜幕，竟无半点星灯。万家灯火皆通明，何时故乡行？

回想三年失意处，不若今夜动人情。

长安夏，一声令，有谁赢？何人安眠何人醒？不过一纸功名。

中元念故人
#

昔时未曾拜祀堂，今日驾鹤还故乡。

三更月满中元夜，你我再叙情一场。

蒙蒙细雨撒苍茫，短暂停泊再启航。

秋风吹散归来客，聚日无多别时长。

军训记
#

我向潇湘君向秦，虽似过客情乃真。

前路漫漫其修远，愿君亦怀报国心。

男儿何必伤离别，我辈岂是蓬蒿人！

文能提笔安天下，武能上马定乾坤。

和鼠头鸭脖讽（原讽为同学所作）
#

  • 鼠头鸭脖讽
    #

    一看你，心事一桩又一桩。

    一看你，悲剧一场接一场。

    不愿再面对空白的墙，只想静想。

    头痛时间愈长，苦闷愈难伪装。

    再能躲藏也要见光，唯有悲伤。

    可悲伤又能怎样，凄凉，可笑也可丧。

莫笑，莫丧，莫悲伤！

古今多少事，从来一样。

寒来暑往，国事兴亡。

善恶中循环，凭依真相。

伤愁一时爽，却无妨。

苦闷凄凉，不如投身报国郎！

我辈青年，何不大干一场？

莫等白头，空梦黄粱。

清明
#

清明时节雨纷纷，路上情人胜旧魂。

试问吾心何处往？若无缘起便封心。

离乡（其一）
#

良辰美景终到头，分分合合几时休？

元月十五宵月夜，借此龙灯驾飞舟。

离乡（其二）
#

聚时有尽别时多，人生何必分分合？

聚时不知别时味，别时才懂聚难得。

默言
#

日月穿梭天天天，人生在世年年年。

一世英名墨寞默，忍辱负重难难难。

江东雪飘无声寒，暮秋叶落竟无言。

何人订立生死簿，敢教日月换新天。

秋雨
#

无边细雨洒朦朦，忽至狂风舞腾空。

细雨不忍红枫落，何必秋风请寒冬？

浓雾
#

前路薄云浓雾，迷醉不知归处。

忽闻耳边厢，似有窃窃嘀咕。

且住，且住，前方禁止进入。

长安游
#

古迹可寻，光阴难在，宫阙楼台追往事。

终南不老，渭水长流，诗文钟鼓入新声。

早发白帝城（改）
#

朝辞彩云白帝行，一日千里还江陵。

猿声不住啼两岸，山过万重已舟轻。

杀秋
#

风急雨骤，旧走新留。

又是一年雨杀秋，数不尽万点零愁。

万般皆从头，过往云烟一并休。

登高楼，争上游，试问鹿死谁手？

胜者王侯，败者寇。

都只落得，四海漂流。

离乡（其三）
#

三更灯火五更鸡，聚时有尽又别离。

天下何处无知己，相望何必泪戚戚。

小马乍行嫌路窄，大鹏展翅恨天低。

只恐人生难尽孝，何时陪伴父母亲？

独留
#

八人寝室七人走，惟留一人守空楼。

别离光景年年有，何必次次哀离愁。

情真无需折杨柳，意切更比奏箜篌。

分别不必多言语，何人相厮到白头？

阵雨
#

来时如电去如风，未等阴时却转晴。

正将撑伞寻庇所，却见风定放光明。

端阳夜雨
#

端阳夜半雨瓢泼，思怀屈子葬汨河。

若非楚君亲奸佞，谁人投河不投国？

雷声点点思绪转，雨中游湖借纸伞。

法海骗饮雄黄酒，却因蛇形惊许仙。

灵芝仙草将命延，却又被骗至金山。

金山寺内水漫漫，不及今夜雨帘天。

高考
#

人生海海路漫漫，十八年来为一天。

皆称人生转折点，命运前途从此变。

人生短短命多舛，高考仅是第一关。

劝君何必太在意，苦尽自然会回甘。

强说愁
#

孤人孤月登孤楼，孤帆孤影驾孤舟。

独言独饮独落寞，独梦独醒独说愁。

知行合一
#

大道至简，亦复何言？

修身克己，敬人自谦。

坚忍格心，明德向善。

欲致良知，必事中练。

净心明理，同时实践。

万事万物，皆有心生。

物我同体，心即自然。

疫情
#

连日阴来今日晴，屋内黯淡屋外明。

疫情网课永不见，还我青春自由行。

情深缘浅
#

一往情深深几许？深山夕照深秋雨。

三世缘浅浅几何？浅池月影浅春鹅。

卷
#

天上一弯新月，地下万点寒光。

蜂拥教室学习忙，却无书声朗朗。

每天身心疲惫，浑身困累难当。

本想打卡操场，谁知排队巨长。

  • 摆——酬卷（来自于室友）
    #

    君不见棠梨花落石板路，晚樱花开染满树。

    君不见池塘春水映斜阳，春分已过日渐长。

    习题未有做完时，青春好景不长留。

    岂因打卡恼心神？操场夜跑又无人。

    学得习，摆的烂，赏青春，谁笑我？

    闲散人如斯，终是难入卷王丛。

    条条大路通罗马，何须再论卷与摆。

    不扰他人功名路，不改此生心归处。

    众生皆在埋头卷，何人听我摆烂言？

宅
#

清明时节连日晴，友人无踪皆出行。

唯有揽卷遣寂寞，无己无功亦无名。

母亲赞
#

素来温文且尔雅，天生丽质显芳华。

生来本是千金体，贤妻良母终为她。

闲来能舞琴棋画，忙时油盐酱醋茶。

名媛才女遍天下，无人能比咱的妈。

吃面
#

一月一人一碗面，亦真亦幻化独缘。

正值孤单萧瑟处，对面无人便无言。

三月春水融山暖，消解三冬寂寞寒。

如此美味无处享，回首不禁泪潸然。

舔狗叹
#

一面之缘，交流甚浅。

万水千山，相谈甚欢。

你情我愿，无人兑现。

笔尖牵动心尖，扶摇直上云天。

我心皈依幻想乡，此生无悔向东方。

定场诗（四十五）
#

天地玄黄宇宙浑，八月十五打了春。 那年事情好奇怪，要提起来吓坏人。 种茄子愣接独头蒜，老窝瓜变成狗头金。 杨八姐游春骑着车，哈哈大笑又泪纷纷。 她一眼哭来一眼笑，哭了声，小白脸儿的丈夫程咬金。 你大破天门阵死得苦，那本是刘伯温害你命归阴。 心中恼怒黄三太，也不该手榴弹炸死潘巧云。 乱七八糟大杂烩，到下面，三堂会审窦尔墩。

定场诗（四十六）
#

远瞧忽忽悠悠，进看飘飘摇摇。 不是葫芦不是瓢，在水中一冲一冒。 这个说像皮球，那个说像水泡。 二人打赌江边瞧，原来是两个和尚洗澡！

定场诗（四十七）
#

浪说曾分鲍叔金，谁人辨得伯牙琴？ 于今交道奸如鬼，湖海空悬一片心。 势利交怀势利心，斯文谁复念知音？ 伯牙不作钟期逝，千古令人说破琴。

定场诗（四十八）
#

手把青秧插野田，低头便见水中天。 六根清净方为道，退步原来是向前。

定场诗（四十九）
#

不是冤家不聚头，冤家相聚几时休？ 早知死后无情义，索把生前恩爱勾。

定场诗（五十）
#

忆昔去年春，江边曾会君。 今日重来访，不见知音人。 但见一抔土，惨然伤我心。

定场诗（五十一）
#

摔碎瑶琴凤尾寒，子期不在对谁弹？ 春风满面皆朋友，欲觅知音难上难。

定场诗（五十二）
#

满帘明月满庭霜，被冷香销拂卧床。 燕子楼前清夜雨，秋来只为一人长。

定场诗（五十三）
#

海鳖曾欺井内蛙，大鹏张翅绕天涯。 强中更有强中手，莫向人前满自夸。

定场诗（五十四）
#

堪羡当年李谪仙，吟诗斗酒有连篇。 蟠胸锦绣欺时彦，落笔风云迈古贤。 书草和番威远塞，词歌倾国媚新弦。 莫言才干风流尽，明月长悬采石边。

定场诗（五十五）
#

得便宜处欣欣乐，不顺心时闷闷忧。 不讨便宜不折本，也无欢乐也无愁。 广知世事休开口，纵会人前只点头。 假若连头俱不点，一生无恼亦无愁。

定场诗（五十六）
#

项托曾为孔子师，荆公反把子瞻嗤。 为人第一谦虚好，学问茫茫无尽知。

定场诗（五十七）
#

得岁月，延岁月；得欢悦，且欢悦。 万事乘除总在天，何必愁肠千万结。 金谷繁华眼底尘，淮阴事业锋去血。 临潼会上胆气消，丹阳县里萧声绝。 到头弱草胜春花，运上精金逊顽铁。 逍遥快乐是便宜，到老方知滋味别。 粗衣淡饭足家常，养得浮生一世爷。

定场诗（五十八）
#

赠君一法决狐疑，不用占龟与祝蓍。 试玉要烧三日满，辨才需待七年期。 周公恐惧流言日，王莽谦恭下士时。 假使当时身便死，一生真伪有谁知？

定场诗（五十九）
#

毁誉从来不可听，是非终久自分明。 一时轻信人言语，自有明人话不平。 祖宗制度至详明，百载余黎乐太平。 白眼无端偏固执，纷纷变乱拂人情。

定场诗（六十）
#

五叶明良致太平，相君何事苦纷更？ 既言尧舜宜为法，当效伊周辅圣明。 排尽旧臣居散地，尽为新法误苍生。 翻思安乐窝中老，先识洛阳杜宇声。

定场诗（六十一）
#

文章谩说自天成，曲学偏邪识者轻。 强辨钨刑非正道，误餐鱼饵岂真情。 好谋己遂生前志，执拗空遗死后名。 亲见亡儿阴受梏，始知天理报分明。

定场诗（六十二）
#

风兮凤兮思故乡，邀游四海求其凰。 时未遇时无所将，何如今夕升斯堂？

定场诗（六十三）
#

夜静瑶台月正圆，清风浙沥满林峦。 朱弦慢促相思调，不是知音不与弹。

定场诗（六十四）
#

朵朵峰峦拥翠华，倚云楼阁是僧家。 凭栏尽日无人语，濯足寒泉数落花。

定场诗（六十五）
#

昔年司马逢童杨，今日俞良际上皇。 若使文章皆遇主，功名迟早又何妨。

定场诗（六十六）
#

山色晴岚景物佳，暖烘回雁起平沙。 东郊渐觉花供眼，南陌依稀草吐芽。 堤上柳，未藏鸦，寻芳趁步到山家。 陇头几树红梅落，红杏枝头未着花。

每日青楼醉梦中，不知城外又春浓。 杏花初落疏疏雨，杨柳轻摇淡淡风。 浮画肪，跃青呜，小桥门外绿阴笼。 行人不入神仙地，人在珠帘第几重？

先自春光似酒浓，时听燕语透帘栊。 小桥杨柳飘香絮，山寺绯桃散落红。 鸯渐老，蝶西东，春归难觅恨无穷。 侵阶草色迷朝雨，满地梨花逐晓风。

定场诗（六十七）
#

每羡鸳鸯交颈，又看连理花开。 无知花乌动情怀，岂可人无欢爱。 君子好逑淑女，佳人贪恋多才。 红罗帐里两和谐，一刻干金难买。

定场诗（六十八）
#

双足立地头顶天，乌云遮盖是枉然。 左手抄起量天尺，右手推出一泰山。

定场诗（六十九）
#

闲向书斋阐古今，生非草木岂无情。 佳人才子多奇遇，难比张生遇莺莺。

定场诗（七十）
#

一尘不染，跳出三界外。 万虑皆空，不在五行中。 扫地不伤蝼蚁命，爱惜飞蛾纱罩灯。 池中有鱼钩不钓，笼中灵鸟放长生。 闷来山前观虎斗，无事林中听鸟鸣。 无忧无虑无烦恼，世态炎凉皆看清。

定场诗（七十一）
#

古来表白多白表，向来情书难抒情。 笑看少年真年少，常与生人话人生。

定场诗（七十二）
#

一吕二赵三典韦，四关五马六张飞。 黄许孙太两夏侯，二张徐庞甘周魏。 神枪张绣与文颜，虽勇无奈命太悲。 三国二十四名将，攒底邓艾与姜维。

定场诗（七十三）
#

一蓑一笠一扁舟，一丈丝纶一寸钩。 一曲高歌一樽酒，一人独钓一江秋。

定场诗（七十四）
#

灵山十万载，佛坐九莲台。 猪八戒七情六欲未丢开，孙行者五行山下苦难挨。 师徒四人朝西走，去取那三藏真经。 心无二意，一路西来。

定场诗（七十五）
#

银鞍照白马，飒沓如流星。 十步杀一人，千里不留行。 事了拂衣去，深藏身与名。 闲过信陵饮，脱剑膝前横。

定场诗（七十六）
#

人生在世天天天，日月如梭年年年。 富贵之家有有有，贫苦之人寒寒寒。 升官发财美美美，两腿一蹬完完完。

定场诗（七十七）
#

有有无无且耐烦，劳劳碌碌几时闲。 人心曲曲弯弯水，世路重重叠叠山。 古古今今多变改，善善恶恶有循环。 将将就就随时过，苦苦甜甜过眼完。

定场诗（七十八）
#

恩重山丘，五鼎三牲未足酬。亲得离尘垢，子道方成就。嗏，出世大因由，凡情怎剖，孝子贤孙，好向真空究，因此把五色金章一笔勾。

凤侣鸾俦，恩爱牵缠何日休？活鬼两相守，缘尽还分手，嗏，为你两绸缪，披枷带杻，觑破冤家，各自寻门走，因此把鱼水夫妻一笔勾。

身似疮疣，莫为儿孙作远忧。忆昔燕山窦，今日还在否？嗏，毕竟有时休，总归无后，谁识当人，万古常如旧，因此把贵子兰孙一笔勾。

独占鳌头，谩说男儿得意秋，金印悬如斗，声势非常久。嗏，多少枉驰求，童颜皓首，梦觉黄粱，一笑无何有。因此把富贵功名一笔勾。

富比王侯，你道欢时我道愁，求者多生受，得者忧倾覆。嗏，淡饭胜珍馐，衲衣如绣，天地吾庐，大厦何须搆。因此把家舍田园一笔勾。

学海长流，文阵光芒射斗牛。百艺丛中走，斗酒诗千首，嗏，锦绣满胸头，何须夸口，生死跟前，半时难相救。因此把盖世文章一笔勾。

夏赏春游，歌舞场中乐事稠，烟雨迷花柳，棋酒娱亲友，嗏!眼底逞风流，苦归身后，可惜光阴，懡儸空回首，因此把风月情怀一笔勾。

定场诗（七十九）
#

清茶竹串春三暖，纸墨梅烟梦五寒。 一捧熙熙如我愿，哪管名利奈何然。 不知云中公卿贵，且养山间草木年。 世事纷争如嚼蜡，观月听风莫等闲。

定场诗（八十）
#

帝王说待我君临天下，许你四海为家。 国君说待我了无牵挂，许你浪迹天涯。 将军说待我半生戎马，许你共话桑麻。 书生说待我功成名达，许你花前月下。 侠客说待我名满华夏，许你当歌纵马。 琴师说待我弦断音垮，许你青丝白发。 面首说待我不再有她，许你淡饭粗茶。 情郎说待我高头大马，许你嫁衣红霞。 农夫说待我富贵荣华，许你十里桃花。 僧人说待我一袭袈裟，许你相思放下。

定场诗（八十一）
#

  • 上九流： 一佛祖，二仙，三圣贤，四官，五公卿，六相，七僧，八道，九庄田；
  • 中九流： 一评书，二医，三卜筮（算命），四棋（棋师），五丹青（画匠），六仕（兵卒），七横（说客），八义（侠客），九打渔；
  • 下九流： 一高台（唱戏），二吹（吹鼓手），三马戏，四剃（剃头），五池子（开澡堂），六搓背，七修脚，八配（配种的），九娼妓。

定场诗（八十二）
#

十里长亭无客走，九重天上现星辰。 八河船只皆收港，七千州县尽关门。 六官五府回官宰，四海三江罢钓纶。 两座楼头钟鼓响，一轮明月满乾坤。

定场诗（八十三）
#

春有百花秋有月，夏有凉风冬有雪。 若无闲事挂心头，便是人间好时节。
